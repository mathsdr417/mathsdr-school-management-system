# Academic Supervisor Execution Checklist

## Lesson Oversight
- [ ] Check lesson notes before teaching begins
- [ ] Observe at least 3 lessons per week
- [ ] Provide written feedback after each observation

## Curriculum Delivery
- [ ] Verify schemes of work alignment
- [ ] Track syllabus coverage
- [ ] Identify learning gaps early

## Assessment Control
- [ ] Vet test and exam questions
- [ ] Ensure marking schemes are prepared
- [ ] Review marked scripts for consistency

## Teacher Support
- [ ] Identify teachers needing support
- [ ] Recommend CPD actions
- [ ] Follow up on improvement plans

## Documentation
- [ ] Update academic supervision records
